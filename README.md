## Installation

Install this tool using `pip`:

    pip install fyb

## Usage

For help, run:

    fyb --help

You can also use:

    python -m fyb --help

## Development

To contribute to this tool, first checkout the code. Then create a new virtual environment:

    cd fyb
    python -m venv venv
    source venv/bin/activate

Now install the dependencies and test dependencies:

    pip install -e '.[test]'

To run the tests:

    pytest
